# Demo Upload Context

This mini project contains two Python scripts.

- `test1.py` is about user profile loading and label formatting.
- `test2.py` is about order totals and discount calculation.

Use cases:

- explain which file is responsible for user profile formatting
- explain how order totals are calculated
- summarize the purpose of the uploaded files
- identify which function applies a discount

Important details:

- `format_user_label` returns a string label like `Alice (admin)`
- `calculate_order_total` multiplies `price * quantity`
- `apply_discount` subtracts a percentage from the total
